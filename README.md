Welcome to my project! In this repository you will find Finn_All-Star.ipynb which 
is a Jupyter Notebook file that contains the code I used to get my results.

Additionally, there is Analyst's Journal.docx which is a journal that I made entries in 
during the duration of this project, so that the viewer may have an understanding of
how I think, act, and proceed during the steps and challenges of this project. 

NOTES:
This program can be altered to accommodate the comparison of any player to any players.
Just place the name the player to compare with others in the future_star string, and
names of players to compare in names[], and adjust any parameters you see fit.

For example:
Checking a player like Wemby against other bigs you might emphasize blocks and rebounds 
with special weight over 3point percentage. You will need to update the names of the given 
graphs according to the data you're comparing.
